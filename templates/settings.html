1

Добавьте предметы для мониторинга

Найдите предметы через поиск ниже. Введите название (например AK-47 Redline) и выберите из результатов. Бот будет следить за этими предметами на Steam Market.

Запустите в тестовом режиме

Сначала используйте ТЕСТОВЫЙ режим — укажите виртуальный баланс (например 5000 рублей). Бот будет мониторить реальные цены, симулировать покупки и продажи, показывать подробные логи — но без реальных транзакций.

3

Настройте Steam API для реальной торговли

Когда убедитесь что стратегия работает — заполните вкладку Steam API и переключитесь в LIVE режим. Бот будет использовать ваш реальный баланс Steam.

4

Настройте Telegram уведомления

Создайте бота через @BotFather, получите токен, укажите ваш Chat ID. Бот будет присылать уведомления о каждой сделке.

5

Следите за логами

На Dashboard в разделе "Логи в реальном времени" вы увидите каждый шаг: проверка цены → сравнение с рынком → расчёт прибыли с учётом комиссии Steam → решение о покупке → статус сделки.

1

Steam Web API Key

Получите ключ на steamcommunity.com/dev/apikey. Нужен для получения информации об аккаунте.

2

Steam Guard (Shared Secret & Identity Secret)

Для реальной торговли нужны секретные ключи Steam Guard. Их можно извлечь из мобильного приложения Steam (файл maFiles). Храните их в безопасности!

Поля shared_secret и identity_secret нужны для подтверждения покупок и продаж.

3

Валюта

Выберите валюту: 5 = Рубли (RUB), 1 = Доллары (USD), 3 = Евро (EUR), 18 = Гривны (UAH).

1

Создайте Telegram бота

Напишите @BotFather в Telegram → /newbot → введите название → скопируйте токен вида 1234567890:ABC-DEF...

2

Получите Chat ID

Напишите боту любое сообщение, затем откройте в браузере: https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates. Найдите "chat":{"id": 123456789} — это ваш Chat ID.

3

Проверьте подключение

После ввода токена и Chat ID нажмите кнопку "📨 Тест" ниже — в Telegram придёт тестовое сообщение.

💡

Как работает стратегия покупки

Бот сравнивает минимальную цену листинга со средней рыночной ценой. Если скидка больше порога (например 17%), рассчитывается чистая прибыль с учётом комиссии Steam 15%.

Формула: прибыль = (цена_продажи × 0.85) - цена_покупки

Минимум 17% — этого достаточно чтобы покрыть комиссию Steam и получить небольшую прибыль. При 17% скидки вы в +2% после комиссии.

📊

Стратегии продажи

По рыночной цене — выставляете по средней цене рынка. Продаётся дольше, но прибыль максимальная.

Рыночная цена −N% — чуть ниже рынка. Продаётся быстрее, прибыль немного меньше.

🧪

ТЕСТОВЫЙ режим

Бот использует виртуальный баланс который вы указываете. Мониторинг, расчёты, логи — всё настоящее. Реальные покупки и продажи не совершаются. Идеально чтобы проверить стратегию перед реальным запуском.

🔴

LIVE (Реальный) режим

Бот использует ваш реальный Steam баланс и совершает настоящие покупки и продажи. Для работы необходимы: Steam логин, пароль, Steam Guard секреты. Используйте осторожно!
